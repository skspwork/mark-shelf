# REQ2.2: 階層切替

[R2.1 — 依存関係の把握](../../要求定義/R2-ドキュメント関係の可視化/R2.1-依存関係の把握.md) と [R2.2 — 参照元の把握](../../要求定義/R2-ドキュメント関係の可視化/R2.2-参照元の把握.md) における「深い依存も辿れる」要求を満たすため、リンクグラフで表示する連結範囲を選択中ノードからの深さで切り替えできる。

## 要件

- 1階層／2階層／3階層／全階層を切り替えられる
- 選択状態はブラウザのローカルストレージに保存し、再読込しても保持される
- 切替は即時反映される
- ユーザーが操作した拡大率はローカルストレージに保持し、ファイル切替・再読込後も同じ拡大率で表示される（未保存時のみ全体が収まるように初期フィット）
