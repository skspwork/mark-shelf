# REQ2.1: グラフ描画

[R2.1 — 依存関係の把握](../../要求定義/R2-ドキュメント関係の可視化/R2.1-依存関係の把握.md) と [R2.2 — 参照元の把握](../../要求定義/R2-ドキュメント関係の可視化/R2.2-参照元の把握.md) を満たすため、現在選択中のドキュメントとその関係にあるドキュメントを有向グラフで表示する。

## 要件

- 選択中ドキュメントを強調表示する
- 参照元が下、参照先が上になるよう矢印が上向きに配置される（breadthfirst レイアウトの Y 反転）
- 矢印はマークダウンリンク記法 `[label](url)` の解決結果と、本文中の自動リンクマッチの両方から生成する（URL パス内のフォルダ名は無視する）
- ノードにカーソルを合わせると、本文中のリンクと同じプレビューポップアップがその場で表示される（選択中ノード自身も対象）
- ノードをクリックするとそのドキュメントに遷移する（リンクタブはそのまま維持）
