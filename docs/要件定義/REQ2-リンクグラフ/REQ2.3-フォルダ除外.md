# REQ2.3: フォルダ除外フィルタ

[R2.1 — 依存関係の把握](../../要求定義/R2-ドキュメント関係の可視化/R2.1-依存関係の把握.md) と [R2.2 — 参照元の把握](../../要求定義/R2-ドキュメント関係の可視化/R2.2-参照元の把握.md) をノイズなく満たすため、用語集など「どこからも頻繁に参照されるフォルダ」をグラフ表示から除外できるようにする。

## 要件

- 除外対象フォルダをチェックボックス式の UI で選択できる
- 除外設定はローカルストレージに保存し、再読込後も保持される
- 除外フォルダ配下のノードとその関連エッジは、グラフから表示対象外になる
- ただし、現在表示中のファイル自身が除外フォルダ配下の場合は、そのビューに限りフィルタを無効化する（グラフが空になる事故を防止）
