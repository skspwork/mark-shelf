# REQ4.1: 戻る／進む

[R1.3 — ナビゲーション履歴](../../要求定義/R1-ドキュメント閲覧性/R1.3-ナビゲーション履歴.md) を満たすため、ドキュメントヘッダ部に戻る／進むボタンを置き、ブラウザの履歴操作と同等の挙動を提供する。

## 要件

- 直前に見ていたドキュメントが存在する場合のみ、戻るボタンを表示する
- 戻った後に前方履歴が存在する場合のみ、進むボタンを表示する
- リンク等で新規にナビゲートしたタイミングで前方履歴はクリアされる（ブラウザと同じ挙動）
- 戻る／進むでのファイル切替は、リンククリックと同じ扱いでタブ状態・プレビュー状態もクリーンにする
