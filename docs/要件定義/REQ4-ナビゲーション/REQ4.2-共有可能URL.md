# REQ4.2: 共有可能 URL

[R1.5 — 共有可能 URL](../../要求定義/R1-ドキュメント閲覧性/R1.5-共有可能URL.md) を満たすため、選択中ファイルを URL に反映し、その URL 単体で同じ状態を復元できるようにする。

## 要件

- 選択中ファイルが変わるとブラウザの URL にファイルパスが反映される（`?file=path/to/foo.md`）
- その URL を踏むと、対応するファイルを選択した状態でビューアが起動する
- ブラウザ履歴を増やさない（`replaceState` 相当の更新）
- ファイルパスは URL セーフにエンコードされ、日本語パスも扱える
