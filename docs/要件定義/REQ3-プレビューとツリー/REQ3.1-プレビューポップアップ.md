# REQ3.1: プレビューポップアップ

[R3.1 — 用語の即時確認](../../要求定義/R3-オンボーディング支援/R3.1-用語の即時確認.md) を満たすため、本文中のリンクにカーソルを合わせるとリンク先ドキュメントの内容をその場でポップアップ表示する。

## 要件

- クリック前に内容を確認できる
- 本文の読みを妨げない位置・大きさで出す。画面幅が狭い環境でもノードや本文を覆い隠さないよう、ビューポート幅・高さに応じて自動的に縮小される
- 即座に消える（離脱・スクロール時）
- 本文中の自動リンク（ファイル名の部分一致）と、明示的に書かれたマークダウンリンク `[label](url)` のどちらもポップアップ対象とする。マークダウンリンクはクリック時にアプリ内のナビゲーションで動作し、外部 URL にならない
- ポップアップ内のリンクをクリックした際は、リンクタブからの導線であっても内容タブに切り替えてから遷移する

## 関連用語

- プレビューポップアップ
