# REQ3.2: ツリービュー

[R1.2 — 大画面表示](../../要求定義/R1-ドキュメント閲覧性/R1.2-大画面表示.md) を満たすため、リポジトリ内のフォルダ／Markdown ファイルをツリー状に表示する。

## 要件

- フォルダを展開／折りたためる
- 折りたたみ状態はローカルストレージに保存され、再読込後も維持される。保存状態が無い初回起動時のみ、トップレベルの README 付きフォルダを展開したデフォルト状態で開始する
- 選択中ファイルの祖先フォルダは自動的に展開され、選択中ノードが視界に収まる
- 選択中ノードを視覚的に強調する
