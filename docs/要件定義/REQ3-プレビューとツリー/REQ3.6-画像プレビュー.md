# REQ3.6: 画像プレビュー

[R1.2 — 大画面表示](../../要求定義/R1-ドキュメント閲覧性/R1.2-大画面表示.md) を満たすため、Markdown 中の画像参照を本文と同じパネル内で描画する。

## 要件

- Markdown の画像記法 `![alt](path)` で指定された画像を本文に埋め込み表示する
- 相対パス・リポジトリ内の絶対パスの両方に対応する
- [REQ1.3 BASE_PATH 配信](../REQ1-ビューア起動/REQ1.3-BASE_PATH配信.md) 下でも画像 URL が正しく解決される
- 対応形式は一般的なラスタ画像（PNG/JPEG/GIF/WebP）および SVG
