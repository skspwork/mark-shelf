# REQ3.5: タブシステム

[R1.2 — 大画面表示](../../要求定義/R1-ドキュメント閲覧性/R1.2-大画面表示.md) を満たすため、同じドキュメントに対する「内容／リンクグラフ／履歴」の表示を同じパネル内でタブ切替できる。

## 要件

- 「内容」「リンクグラフ」「履歴」のタブを切り替えられる
- タブ切替はファイル切替より軽く、同じファイルを見ている文脈を保つ
- [REQ4.1 戻る／進む](../REQ4-ナビゲーション/REQ4.1-戻る進む.md) によるファイル切替時は、タブ状態とプレビュー状態をクリーンな初期状態に戻す
- [REQ2.1 グラフ描画](../REQ2-リンクグラフ/REQ2.1-グラフ描画.md) 内のノードクリックでは、リンクタブの状態を維持したまま遷移する
