# SPEC3: UIコンポーネント

[REQ3 — プレビューとツリー](../../要件定義/REQ3-プレビューとツリー/README.md) と [REQ5 — ファイル検索](../../要件定義/REQ5-ファイル検索/README.md) の実装仕様。

- `<PreviewPopup>` — リンクホバー時の本文ポップアップ
- `<TreeView>` — 左ツリー
- `<TableOfContents>` — 本文右の目次
- `<DetailPanel>` — 内容・履歴・リンクのタブ統合
- `<Markdown>` — Markdown レンダリング（GFM / Mermaid / 画像）
- `<SearchBar>` — 全文検索入力
- パネルリサイザー（ツリー幅・目次幅）
- 画像プレビュー（`<img>` の asset 書き換え）
